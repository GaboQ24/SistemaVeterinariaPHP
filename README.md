# Sistema de Gestión Veterinaria

Aplicación web desarrollada en PHP para la administración de una veterinaria.

## Funcionalidades

* Registro de vehículos
* Gestión de empleados
* Registro de citas
* Gestión de ventas
* Sistema de mensajería
* Consulta de información mediante paneles administrativos

## Tecnologías utilizadas

* PHP
* HTML5
* CSS3
* JavaScript
* MySQL

## Características

* Operaciones CRUD
* Gestión de registros
* Interfaz web para administración
* Organización modular mediante archivos de inclusión
